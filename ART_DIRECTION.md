# Art Direction — "clean stylized indie" (the standard, not vibes)

Derived from real reference (Firewatch, Content Warning) after the game looked "too neon."
**Apply this checklist to every render before calling it good. When in doubt, REMOVE, don't add.**

## The core lesson
Quality in this style comes from **matte surfaces + restrained palette + value structure +
atmospheric depth + motivated light** — NOT from glow, saturation, or detail. Low-poly looks
great when lit and graded well. Emissive glow and saturation boosts are what make it look amateur.

## Rules (check each render against these)
1. **Nothing glows except actual light sources.** Lamps, screens, the sun, runway lights — yes,
   restrained. Walls, columns, floors, signs, trim — **MATTE**, lit by the environment. (My #1 past
   mistake: glowing architecture = neon.)
2. **Palette = ~2 temperatures**: a warm key light + cool shadow/ambient. Pick ONE accent hue, used
   sparingly. No more than ~4-5 hues in frame.
3. **Desaturate.** Pull saturation DOWN. Muted beats punchy. Grade saturation ≤ 1.0 (often 0.9-0.95).
   Accent colors should be muted (e.g. teal-grey, not pure cyan; ochre, not pure amber).
4. **One small saturated accent** for a focal pop (Firewatch's red compass). Everything else muted.
5. **Value hierarchy + atmospheric depth.** Foreground darker, background lighter; distance fades to
   the cool sky/haze color (fog). This makes depth and mood far more than glow does.
6. **Motivated lighting.** One clear key (sun/skylight) + soft fill. Let shadows + AO define form.
7. **Glow/bloom subtle.** Low intensity, HIGH hdr threshold (only the brightest lights bloom).
8. **Contrast natural** (~1.0). Don't crank it.

## Render-review process (the "refined eyes" loop)
- Render the REAL GPU shot (RTX 3070 loop), HUD hidden.
- Compare side-by-side with a reference image of the target look.
- Write a HARSH critique first (what's wrong: too bright? glowing? too saturated? flat/no depth?
  too many hues?) BEFORE noting what works.
- Only ship when it passes the checklist. Then still defer final taste to Noam — he's art director.

## Past mistakes to not repeat
- Treated emissive glow as "polish" → neon overload (teal column strips, glowing wayfinding/signage).
- Boosted saturation/contrast in the color grade → punchy/garish.
- Declared "looks good" from a thumbnail with no standard. Use this checklist + reference instead.
