Drop your squad photos here (.png / .jpg / .jpeg / .webp).

They auto-hang as framed, spotlit banners around the airport terminal in the
lobby — no code changes needed. If there are more wall slots than photos, the
photos cycle to fill them.

There are 11 banner slots in the current lobby (5 back wall, 3 left, 3 right).

You can also drop a `banners/` folder next to the game .exe at runtime and those
images will be picked up too.
